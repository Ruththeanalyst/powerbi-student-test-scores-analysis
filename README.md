# powerbi-student-test-scores-analysis
Power BI dashboard analyzing student reading and math performance across districts, states, and activities

## Project Overview
This project uses Power BI to analyze student academic performance data. 
The dashboard focuses on reading and math scores across districts, states, grades, 
and extracurricular activities to support data-driven educational decisions.

## Dashboard Insights
- Average reading score by district
- Average math score by extracurricular activities
- Impact of math tutoring periods by grade
- State-level reading score averages
- Year-over-year percentage change in reading scores

## Tools Used
- Power BI Desktop
- DAX for calculations
- Data modeling and visualization techniques

## Value of the Project
This dashboard helps educators and decision-makers:
- Identify districts with lower performance
- Understand how extracurricular activities affect math scores
- Track improvements or declines over time
- Make informed decisions on academic support and interventions
